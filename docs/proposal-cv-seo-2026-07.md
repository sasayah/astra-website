## 4. サイトコンテンツ最適化（CV向け）

### 4-1. フォームの必須バリデーション（実装バグ級・即対応推奨）

`/contact` のメインフォームは現在**全項目空でも送信できる**（旧WPの検証JSが新環境で無効・API側も未検証）。空問い合わせ・bot送信はCV計測ノイズにもなる。
→ 電話番号を必須化 + 最低限のクライアント検証を実装（新LPのフォームは実装済みパターンあり）。

### 4-2. /ihinseiri（遺品整理）のLP化改善

遺品整理 完全一致に全期間¥24,067投下してCV0。現状ページは情報羅列でCTAが弱い。
→ 新LPテンプレート（/lp/と同型）で: ファーストビューに電話CTA・「市役所で紹介中」・女性スタッフ在籍・間取り別料金目安（1K¥35,000〜等、実在の訴求のみ）・遺品整理特有の安心訴求（合同供養・買取・許可番号）を配置。改善までは広告予算を絞る（3-4）。

### 4-3. 電話CTA最優先の徹底

電話CV¥10,000 vs LINE/フォーム¥3,000。現LPは電話・LINE・フォームが等列。
→ スマホでは**追従バーの電話ボタンを最大化**（現構成でも対応済みだが、ファーストビュー直下にも電話ボタンを重複配置）。深夜帯はLINE/フォームを上位に出す時間帯出し分けも検討価値あり（24時間受付を訴求しているため現状でも矛盾はない）。

### 4-4. エアコン・テレビLPの訴求修正（CPA悪化中の2品目）

- エアコン: クエリは「取り外し」「取り外し 料金」「撤去」が高CVR（全期間CPA¥2,437〜¥4,926）。/lp/aircon/ の見出しを「処分」中心→**「取り外し・撤去込み」を主訴求**に変更。
- テレビ: 「引き取り」クエリが優良（CPA¥3,495）。ブラウン管CVR低。**家電リサイクル法で粗大ごみに出せない**ことを冒頭で明示し「引き取り」語を強化。

### 4-5. スマホUX微修正

- メニューのサブメニュー開閉「＋」が13×13px → 44px相当に拡大（行全体をタップ可能に）。
- 未分類/バグ修正済み: slick未ロードエラー（修正済み）、日本語URL404（修正済み）。

## 6. SEO対策（第二目的）

1. **一覧ページのnoindex解除**: `/blog` `/voice` `/news` の一覧が旧サイト設定のままnoindex。ナビから張られている主要ページであり、**特に /blog 一覧のnoindexは208記事へのクロール動線を殺している**。解除推奨（manifest.jsonの3行変更）。
2. **重複タイトルの解消**: `/huyouhin` `/huyouhin2` `/huyouhin3` `/huyouhin-line` `/huyouhin-ph` `/huyouhin2-*` がトップと同一title・全てindex可能 → カニバリ状態。広告専用ページは**noindex化**（下記7と連動）。
3. **kansai-huyouhin 124ページの活用**: 地域×不用品回収の自然検索の主力。品目LPからの内部リンク・逆リンクで相互強化。タイトルは「{市}の不用品回収」で最適化済み。
4. **構造化データ**: LocalBusiness（住所・電話・営業時間・エリア）+ FAQPage（/faq、LPのFAQ部）を追加。
5. **ガイド記事**（上記5）: 品目×「処分方法」informationalクエリでの獲得 → LPへ送客。
6. 技術面は対応済み: sitemap修正（396 URL・日本語URLエンコード）、Next.js化によるCore Web Vitals改善見込み。切替後にSearch Consoleで計測。
