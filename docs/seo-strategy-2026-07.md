### 検索クエリ側の機会（広告データから逆算）

広告で実証済みの商業クエリ＝SEOで獲れれば広告費ゼロでCVになるクエリ:

1. **地域×不用品回収**（東大阪 CVR36%、大阪市 CVR26%）→ kansai-huyouhin の守備範囲
2. **品目×処分/引き取り**（冷蔵庫処分 17,930imp、マットレス処分、タンス処分）→ 現状受け皿なし＝ブログガイド記事の狙い目
3. **大阪市 粗大ごみ**（40,000imp/全期間）→ 行政情報コンテンツの狙い目（自治体公式が強いが、「出せない品目・運び出せない人」の受け皿になれる）
4. エアコン取り外し系 → 同上

## 2. SEO戦略の3原則

1. **刈り取りは広告、SEOは「実証済みクエリの無料化」**。広告で CVR が確認できたクエリ群だけに
   SEOリソースを投下する（当てずっぽうのコンテンツを作らない）。
2. **1クエリ群=1URL**。品目×地域の商業クエリは kansai-huyouhin、品目×情報クエリはブログ記事、
   広告はLP。相互にカニバらせない（LP群はnoindexで隔離済み）。
3. **実在の一次情報だけで書く**（実在価格・実事例・実写真）。価格表や事例は既に原典があるので、
   これをクロール可能なテキストで各層に展開する。

## 3. サイト設計（SEO視点の情報アーキテクチャ）

```
トップ（指名・ブランド）
├─ /kansai-huyouhin（エリアハブ）
│    └─ {市区}×123 …「{市} 不用品回収」商業クエリの受け皿
│         今回強化: 品目別価格表(テキスト) + リサイクル法解説 + FAQ(構造化データ) + パンくず
├─ /blog（更新性・E-E-A-T）
│    ├─ 作業事例 209記事 … タイトル一意化済み「{市}の不用品回収事例（日付）」
│    └─ [新設予定] 品目別処分ガイド …「{品目} 処分」情報クエリの受け皿
├─ /service /huyouhin /ihinseiri /faq /company /voice … ブランド・指名補完
└─ /lp/**（noindex・広告専用・SEO不参加）
```

- 内部リンク: トップ/huyouhinから全市区ページへのリンクは既存（各118本）。
  今後はガイド記事→市区ページ→contactの動線を張る。
- 地域ページの検索意図カバー: 「料金いくら？」「家電は出せる？」「すぐ来て」を
  今回追加したセクションで全て回答（滞在・直帰の改善も狙う）。

## 4. コンテンツ計画（P4で実施）

### 4-1. 品目別処分ガイド記事（5本、優先度順）

広告実績と検索ボリュームの交点で選定。各1,500〜2,000字+実在価格+市区ページ/LPへの内部リンク:

1. 冷蔵庫の処分方法（大阪版）— 家電リサイクル法・自治体で出せない理由・費用比較
2. マットレスの処分方法 — スプリング入りが粗大ごみで断られるケース
3. タンス・婚礼家具の処分方法 — 運び出せない問題・買取相殺
4. エアコンの取り外しと処分 — 取り外し費用の相場・無料回収の注意
5. 大阪市の粗大ごみの出し方と代替手段 — 手数料・申込方法・家電4品目の扱い

※ブログ一覧は静的HTMLのため、記事追加時に content/pages/blog/index.html への導線追記もセットで行う。

### 4-2. 地域ページの段階強化

- 第1弾（実装済み）: 全123ページ共通の価格表+FAQ+構造化データ
- 第2弾: 高CVR市区10ページに市区固有情報（その市の粗大ごみ手数料・受付センター等）を追記。
  **行政情報は必ず各市公式サイトで事実確認してから記載**（誤情報リスクがあるため自動生成しない）
