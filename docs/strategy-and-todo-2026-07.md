## 2. 戦略の柱（見直し版）

### 柱1: 本番切替を最優先（今週中目標）

これが終わるまで柱2以降は動かせない。チェックリストは §7 P1。

### 柱2: 「価格開示」ポジションへの転換 ← 今回の見直しの核

初版提案は「訴求（安心・即日）で選ばれる」前提だった。実在価格の確認により
「**価格で選ばれる**」戦い方が可能になった。ただし一気に切り替えず、ABテストで検証する:

- **A案LP**（訴求型・価格なし） vs **B案LP**（価格ファースト・品目別料金表）
- 勝敗が付いたら、広告文にも価格を展開（例:「冷蔵庫回収6,000円〜｜追加料金0円」）。
  価格入り広告文はCTRを下げてもクリックの質（価格了承済み）を上げ、CPA改善が見込める。
- 電話CV（価値¥10,000、フォームの3.3倍）最優先の導線はA/B共通で維持。

### 柱3: 予算の外科手術

大型一般（45%消費・ROAS53%）を解体し、好調単品（タンス・ソファ・冷蔵庫・マットレス）と
高CVR地域クエリに再配分。目標CPA ¥8,500（全期間平均水準への回帰）、
その後ROAS100%超え。

### 柱4: 二段構えのSEO

広告LPはnoindexを維持（検証の場）。**広告で勝った型・コピー・価格表示を
インデックス層（kansai-huyouhin 124ページ / ブログ）へ還元**する。
先にSEOページを作り込まない（勝ちパターン確定前の作り込みは手戻りになる）。

## 3. サイト設計（3層アーキテクチャ）

```
┌─ 広告LP層（noindex・実験場）───────────────────┐
│ A案 /lp/{item}[/{city}]     訴求型             │
│ B案 /lp/b/{item}[/{city}]   価格型 ←ABテスト   │
│ 旧LP /huyouhin2系 → 移行後301で削除            │
└──────────────┬──────────────────────────────┘
               │ 勝ちパターンを還元
┌─ SEO層（index・資産）─────────────────────────┐
│ /kansai-huyouhin/{市区} ×124  地域×回収の主力  │
│ /blog ×210 + /未分類 ×37     記事資産          │
│ 強化: 行政粗大ごみ情報・価格表・事例・FAQ       │
└──────────────┬──────────────────────────────┘
               │ 内部リンク
┌─ ブランド層（index）──────────────────────────┐
│ / /service /huyouhin /ihinseiri /faq /company  │
│ /contact（フォーム）/voice                      │
└───────────────────────────────────────────────┘
```

- 広告LP層のフォームは hidden `lp` 値（B案は `B:` プレフィックス）で流入元を判別可能。
- 層をまたぐ重複title・カニバリは解消済み（旧LP群noindex化）。

## 4. ABテストの設計（切替後すぐ開始）

| 項目 | 内容 |
|---|---|
| 対象グループ | CV量のある4品目: 冷蔵庫・タンス・ソファ・マットレス（他は様子見） |
| 方法 | 各広告グループに同文言・最終URLのみ違う広告を2本（A行き/B行き）。広告の最適化配信をオフ（均等ローテーション）に設定 |
| 主要指標 | CVR（電話+LINE+フォーム合算）とCPA。フォームは `lp` 値で判別、電話/LINEはLP別に分離できないため**広告(=URL)単位のCV列**で比較 |
| 判定基準 | 各腕クリック150以上 または 2週間経過 かつ 合算CV10件以上。差が僅少なら電話CV率を優先 |
| 勝ち処理 | 勝者に全品目のURLを寄せる → 次のテスト（広告文への価格挿入 vs なし）へ |

※Yahoo!検索広告にはA/B分割機能がないため「同文言・URL違い・均等配信」で代替する。
